# Akvelon RPG SAGA

Учебный проект на TypeScript. Игра-турнир: задаётся количество героев,
дальше игра идёт сама, а пользователь смотрит лог сражений.

## Запуск

```bash
npm install
npm run play               # сборка + запуск (8 игроков, задержка 700 мс)
node dist/index.js 6       # 6 игроков
node dist/index.js 6 1000  # 6 игроков, задержка 1000 мс между строками
```

Лог сразу целиком сохраняется в файл `game.log`, а в консоль строки
выводятся по очереди с задержкой — чтобы человек успевал читать.

## Файлы

- `Hero.ts` — базовый класс героя (наследуются Рыцарь, Лучник, Маг)
- `Knight.ts`, `Archer.ts`, `Mage.ts` — типы героев и их способности
- `HeroFactory.ts` — фабрика героев (два фабричных метода)
- `Logger.ts` — вывод логов
- `Game.ts` — игровой процесс (турнир)
- `index.ts` — точка входа
